# Contributing to Learning Bot

Thank you for your interest in contributing to Learning Bot!

## Getting Started

1. Clone the repository
2. Install dependencies (see README.md)
3. Create a feature branch
4. Make your changes
5. Test your changes
6. Submit a pull request

## Development Setup

```bash
# Clone the repository
git clone https://github.com/momfy11/Learning_bot.git
cd Learning_bot

# Install dependencies (adjust based on your stack)
pip install -r requirements.txt  # For Python
# or
npm install  # For Node.js
```

## Code Style

- Follow the existing code style in the project
- Write clear, descriptive commit messages
- Add comments where necessary

## Testing

Make sure to test your changes before submitting a pull request.

## Questions?

Feel free to open an issue if you have any questions!
